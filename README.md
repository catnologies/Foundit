# Foundit
Foundit brings fast, in-place search to macOS Finder. Search the current Finder folder without opening a new tab or leaving your workflow. Launch with a shortcut, type a file or folder name, and jump through matching results instantly. Lightweight, fast, and built to feel native on macOS.

## How it works

1. Open a Finder folder.
2. Click Foundit’s magnifying-glass menu-bar icon, or use your shortcut.
3. Type part of a file or folder name.
4. Press Return, or use the up/down arrows, to move through matching files.

## Features

- Searches only the Finder folder currently in use
- Highlights matching files directly in Finder
- Custom keyboard shortcut
- Light, Dark, and Auto appearance
- Optional menu-bar icon hiding
- Lightweight: no background folder scanning

## Why Foundit?

Finder search can open a separate search view or tab. Foundit keeps you in the folder you are already browsing, so finding the next file is quick and distraction-free.
